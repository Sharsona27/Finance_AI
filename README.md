Finance_AI
AI-Powered Finance Assistant.A smart full-stack personal finance dashboard. This web app helps users manage income, expenses, and financial goals with the help of an AI Assistant and speech-based inputs.

Key Features

Income & Expense Tracking– Easily track daily income and expenses
AI-Powered Financial Assistant– Get smart insights and suggestions
Goal Management– Set and monitor financial goals
Smart Budget Planning– Personalized budget recommendations
Speech-Based Input– Add expenses using voice commands
Visual Dashboards– Clear charts and summaries for better understanding
Secure User Authentication– Safe login and data protection
